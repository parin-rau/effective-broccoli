# Project Management App

## Basics

-   Adapt existing project app for Remix JS
-   Remove groups section
    -   i.e. only projects and tasks
-   Per user tasks
    -   no sharing with other accounts, at least to start
-   frontend, backend, db hosted on RPi

## Tech

### Frontend

-   React
-   Tailwind

### Backend

-   Remix
-   Express/Node
-   Prisma
-   PostgreSQL

### Deployment

-   Docker
-   Nginx
-   RPi
-   Free online hosting (?)
